# Shopping Cart with React and API Rest

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

# Run this app

### cd db products
## `npm install -g json-server`

## ``json-server cart.json``

### cd cart
## `npm run dev`
